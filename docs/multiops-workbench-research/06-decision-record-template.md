## 10. Validation Before Implementation

- [ ] 关键事实有可追溯来源
- [ ] 高影响假设有验证结果或明确风险接受
- [ ] 至少两个可行替代方案已评估
- [ ] 安全、权限和外部副作用经过独立评审
- [ ] 上游合并影响已评估
- [ ] 验收命令和失败路径明确
- [ ] 用户/决策人明确批准

## 11. Revisit Triggers

- 第二个真实 Connector 出现
- 使用量或失败率达到阈值
- Multica 上游相关模块发生重大变化
- 外部框架许可证或产品边界变化
- 运行成本超过预算
- 关键假设被真实数据推翻
